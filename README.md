# hi_world

This is my first published module and says "Hi world!"

## Installation

<!-- The first step is adding the modelu in your Gemafile -->

With `yarn`

```bash
yarn add hi_world
```

With npm

```bash
npm install hi_world
```

## Usage

Import the module

```js
const hiWorld = require("hi-world");
```

You will have available the function `hiWorld`.

```js
hiWorld();
```
